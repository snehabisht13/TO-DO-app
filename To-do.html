To-Do App

  1. Enter "list" to show the list.
  2. Enter "add" to add in list.
  3. Enter delete to remove from list.
  4. Enter "quit" to quit app.

Instructions

  • Open console first.
  • If you're done adding and listing tasks write "quit" command & the app will be closed.
  • if you want to enter more task you'll have to rewrite "add" command after each task is added.
  • For deleting any task write index carefully.
